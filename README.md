Privacy Policy

This extension is designed to assist users in securely transferring session information from authorized websites to user-controlled external services for legitimate workflow integration purposes.

Data access

The extension accesses authentication cookies only from the domain:
https://support.fortinet.com

This access is strictly limited to retrieving the current authenticated session state required for functionality.

Data usage

Cookie data is used solely to enable session continuity between the authorized website and user-specified trusted applications.

Data transmission

No data is sold, shared, or distributed to third parties.

Data is transmitted only to user-controlled and explicitly allowed endpoints:

http://localhost:3000
https://*.fortilab.net
Data storage

The extension does not permanently store any cookies or authentication data. All processing occurs in memory and data is immediately discarded after transmission.

User control

Users have full control over data flow and can disable the extension or revoke browser sessions at any time.

Security

All communication is limited to secure or user-controlled environments. The extension does not modify authentication data or interfere with website behavior.
